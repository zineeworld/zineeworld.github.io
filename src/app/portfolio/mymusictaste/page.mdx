# MyMusicTaste
- 2015.06 ~ 2018.01
- 주요 서비스 : 데이터 기반 수요 예측 공연 기획/제작
- 기술 스펙 : HTML, Sass, Django, Backbone.js, Vue.js, Github

---

재직하는 동안 스타트업의 모든 점을 보고 느끼고 배울 수 있었던 곳입니다. 처음 입사했을 때는 15명 남짓이었는데 퇴사할 때는 50명 이상의 규모가 되었고 2019년 말에는 130명이 넘는 조직으로 성장하기도 했었습니다. 시리즈 투자를 계속 받아가면서 성장해 나가는 회사에서 함께 프로덕트의 처음부터 런칭까지 함께 할 수 있어서 뜻깊은 순간으로 기억에 남습니다.

다양한 기획자, 디자이너, 개발자 모두 함께 하나의 프로덕트를 만들기 위해 고군분투 했던 시간들이 참 따뜻한 기억으로 남아있습니다. 끊임없이 개인의 성장에 대해 관심을 갖고 챌린지를 주었던 곳입니다. 웹 퍼블리셔에서 UI 개발자라고 부를 수 있을 정도의 업무를 전담하게 되었고 2015년 6월부터 `Sass`를 쓰기 시작했습니다. 서비스가 `Django` 기반이어서 마크업 산출물을 별도로 관리하였습니다. 마크업 변경 사항이 생기면 프론트엔드 개발자가 마크업 산출물 diff 확인 후 수정하는 방식으로 일했습니다. 버전관리는 `github` 을 사용했습니다.

입사 2년간은 개발팀에서, 그 후 7개월은 UX팀에서 일하면서 업무 스펙트럼을 넓게 가질 수 있었습니다. 끝무렵에는 PM 업무를 맡아주기를 원하셨으나 마크업 개발자가 더욱 필요한 곳에 추천을 받게 되어 이직하게 되면서 떠나게 되었습니다.
