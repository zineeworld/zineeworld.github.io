  # 내부 운영 툴 개선 기획/디자인/마크업

  2017년 4월에 진행했던 내부 툴 개선 프로젝트에서 프로젝트 문서 관리, 기획, 와이어프레임, 디자인, 마크업을 맡았습니다. 현재 운영 툴은 Django에서 자동으로 제작되는 형태로, 백엔드 개발자분이 만들어 주시는 방식이라 UI/UX 적인 측면을 고려하기보다는 사용하는 사람이 빠르게 적응하는 데 중점을 두고 있습니다. 워낙 많은 툴이 존재하기 때문에 하나씩 개선해 나가는 것을 목표로 진행하였으며, 다른 과제들의 우선순위에 밀려 3가지 정도의 툴만 개선한 후 일시정지된 상태입니다.
